# DSA-tutort-session

This repository contains Data Structures and Algorithms tutorial exercises.

## Setup

1. Make sure you have Node.js installed (version 14 or higher)
2. Install dependencies:
   ```bash
   npm install
   ```

## Running TypeScript Files

To run a TypeScript file, use the following command:

```bash
npx ts-node <path/to/file.ts>
```

For example, to run the Move Zeroes assignment:

```bash
npx ts-node Array/ArrayAssignmnet1/MoveZeroes.ts
```

This will execute the TypeScript code and display the output in the terminal.
